# Guard Tracking System

## Overview

This project is a mini guard tracking system built using Spring Boot and JavaScript.

## Features

* Check-in / Check-out with timestamp and location
* Admin dashboard to view attendance
* Attendance logs with total working hours
* Mobile-friendly interface
* REST APIs

## Tech Stack

* Backend: Spring Boot (Java)
* Frontend: HTML, CSS, JavaScript
* Storage: Excel (Apache POI)

## How to Run

1. Run Spring Boot application
2. Open:

   * http://localhost:8080/index.html
   * http://localhost:8080/admin.html

## APIs

* POST /api/checkin
* POST /api/checkout
* GET /api/logs
* GET /api/status
